# Student Management API

A simple RESTful API built using **Node.js, Express, and MongoDB** for managing students.

---

## 🚀 Features
- Add new students
- Get all students
- Get student by ID
- Update student details
- Delete student

---

## 🛠️ Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
